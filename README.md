# Pandasfile
Pandas Series is a one-dimensional labeled array that can store any data type. It behaves like a single column of data with an index for each value.
# What is a Pandas Series?

A Series contains:

* Data → actual values

* Index → labels for each value
# Features of Series

* One-dimensional data structure

* Each item has a unique index

* Fast operations

* Supports NumPy-like calculations

* Can store any data type (int, float, string, object)
# Reading Data into Series

You can create a Series from:

* Lists

* Dictionaries

* NumPy arrays
# Use Cases

* Data cleaning

* Data preprocessing

* Storing one-dimensional datasets

* Machine learning input features

* Time-series data
